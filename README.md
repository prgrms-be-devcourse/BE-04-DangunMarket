- #### 김별
  - 게시글 
  - 로그인
  - 회원(활동지역)
  - 채팅
- #### 윤영운
  - 게시글 
  - 지역정보 세팅
  - 채팅
- #### 김민희
  - 게시글
  - S3 이미지 관리 및 저장 
  - 채팅
